Sororite Vintage App (2022)

Language: Swift
Tools: XCode View on GitHub

Overview

The app provides users with a portable way to shop luxury pieces from the online website Sororite Vintage (linked below), which is contains a multitude of purchasable vintage pieces. The app is compatable with any iOS device.

View Website

Implementation

In order to transfer the functionality of the website into an app-friendly version, I simply relied on the WebKit UI functionality that comes with XCode. This allows for data fetching from a specified URL, which I then translated into displayable UI data. This way, the mobile app is perfectly the same as the website, since it does not use any of its own features and serves more as an information fetcher. This prototype can be used for any website, simply by replacing the URL.

Basic Usage

The usage of this app is precisely that of the online website, as it possess all of the same functionalities.

In-App Views

The app's user interface is resized to fit the screen. All design is merely a duplicate of what is displayed on the online website.
