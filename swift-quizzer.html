Swift Quizzer (2022)

Language: Swift
Tools: XCode View on GitHub

Overview

A simple iOS app that allows users to learn the basics of Swift programming and SwiftUI through detailed videos and quizzes to test their knowledge.

Implementation

The app fetches content from both locally and remotely hosted JSON data in order to display it for lessons and quizzes within the app. The remote data is hosted through github pages, which allows for mutabilty - whenever the JSON data is altered on the website, the content displayed in the app gets altered as well.

Basic Usage

Users of this app are able to select whether they would like to learn Swift or SwiftUI. Upon clicking on their preffered model, users are then taken to a list of lessons that need to be completed to master the topic. Each lesson contents a short instructional video, as well as a text description. Further, in order to asses their knowledge, users can take a final quiz and see their results.

In-App Views

The app's user interface is designed with Figma and graphics are from Pinterest and Google Images. Content used within the app was sourced from Google as well.
